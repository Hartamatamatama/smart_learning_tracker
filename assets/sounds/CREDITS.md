# Kredit Ambient Sound

Semua file di folder ini diunduh dari **Mixkit** (https://mixkit.co/free-sound-effects/),
yang menyediakan sound effect gratis untuk dipakai tanpa biaya. Dokumentasi ringan ini
untuk keperluan tugas kuliah internal (tidak dipublikasikan).

| File | Nama Suara | Kategori | Sumber URL |
|------|-----------|----------|------------|
| `rain.mp3` | Rain long loop | Hujan | https://mixkit.co/free-sound-effects/rain/ |
| `forest.mp3` | Forest birds ambience | Alam / Hutan | https://mixkit.co/free-sound-effects/ambience/ |
| `cafe.mp3` | Restaurant crowd talking ambience | Kafe / Keramaian | https://mixkit.co/free-sound-effects/ambience/ |
| `crickets.mp3` | Crickets and insects in the wild ambience | Malam / Serangga | https://mixkit.co/free-sound-effects/ambience/ |
| `campfire.mp3` | Campfire night wind | Api unggun / Angin | https://mixkit.co/free-sound-effects/ambience/ |

Format: MP3 128 kbps, 44.1 kHz. Semua di-loop otomatis oleh aplikasi (just_audio,
`LoopMode.one`) sehingga terdengar menerus selama sesi belajar.
